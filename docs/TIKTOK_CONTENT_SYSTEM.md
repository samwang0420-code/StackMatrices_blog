## 📝 工作流程

### 1. 生成脚本
```bash
python3 scripts/generate_tiktok_script.py
```

输出:
- 脚本 (用于配音)
- Visual Notes (画面建议)
- Caption (TikTok文案)

### 2. 录音
- 用微软TTS朗读脚本
- 或自己录音

### 3. 制作视频
**推荐工具**:
- CapCut (手机/电脑)
- Canva (在线)

**模板**:
1. 导入生成的图片
2. 按脚本顺序排列
3. 每张图停留时间 = 对应脚本秒数
4. 添加配音
5. 添加文字overlay
6. 添加背景音乐

### 4. 发布
- 手动上传到TikTok
- 复制脚本中的Caption
- 添加Hashtags

## 🎯 内容策略

### 黄金公式
```
Hook (0-2秒) → 痛点/数据震惊
Body (2-20秒) → 解决方案/案例
CTA (最后5秒) → 引导评论/关注
```

### 成功的关键
1. **前2秒抓住注意力** - 用数据/痛点
2. **每3秒一个信息点** - 不要拖
3. **结尾强CTA** - "评论区扣X"
4. **统一风格** - 每次结尾一样
